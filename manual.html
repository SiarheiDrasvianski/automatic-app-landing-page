How to use MNQN app

 

Tap menu and select layer image

 

Place the image so it approximately matches mannequin scale and position

Pin the camera position, so you can recall it later

Toggle mannequin style in menu if you need more contrast, and start mimicing the pose by dragging manneguin limbs

Rotate mannequin if needed for better editing - and recall original camera position if needed

Disable image layer to see result

Now change the camera position to desired so you can use this pose as a reference. Select snapshot sharing in menu to import it to your drawing app, or just press Home+Lock to make screenshot.

The pose can be saved to file for later use or sharing.
